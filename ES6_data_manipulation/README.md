# 📊 ES6 Data Manipulation

## 📖 Description

Ce projet introduit et pratique la manipulation de données en **JavaScript ES6**, à travers l’utilisation des **Tableaux (map, filter, reduce)**, des **Typed Arrays**, ainsi que des **structures de données Set, Map et WeakMap**.

Il est conçu pour consolider la logique fonctionnelle de JavaScript et améliorer la manipulation de données en mémoire.

---

## 🎯 Learning Objectives

À la fin de ce projet, je serais capable d’expliquer :

- Comment utiliser `map`, `filter` et `reduce` sur les tableaux
- Le concept des **Typed Arrays**
- L’utilisation des structures de données **Set**, **Map** et **WeakMap**

## 📋 Tâches

0. Basic list of objects → Retourner une liste d’étudiants sous forme d’objets

1. More mapping → Extraire uniquement les IDs des étudiants

2. Filter → Filtrer les étudiants par localisation

3. Reduce → Calculer la somme des IDs des étudiants

4. Combine → Mettre à jour les notes des étudiants par ville

5. Typed Arrays → Créer et manipuler des Int8Array

6. Set data structure → Générer un Set à partir d’un tableau

7. More set data structure → Vérifier si un tableau est inclus dans un Set

8. Clean set → Extraire des valeurs de Set selon un préfixe donné

9. Map data structure → Créer une liste de courses avec un Map

10. More map data structure → Mettre à jour certains éléments d’un Map
